Synthesthesia

I've always loved music, and I've produced music for fun since my late teens.
So I thought, why not combine my love for music and programming to create a digital synthesizer?
I've chosen to code it in Rust as I think it's a fascinating language, which I want to get better at.
I'm calling it Synthesthesia and currently it doesn't have a lot of functionallity. You can
read about the specifics of how far I've gotten in the Github repository linked below!
