To Do List Application

For a long time now, I've been wanting to get into React and learn how to build apps,
and so i thought: what better way than creating a simple To Do List App to learn the framework?
The application is very simple: The user writes a Task using the input field, and adds it by
pressing the plus button. The user can the complete the Task by pressing it, and it disappears.

The code is quite simple, most of it being styling. The main and only component is a Task, which is
just a styled View consisting of a little pin symbol and some given Text. The app contains an array of
these Tasks, and has a method for removing them, which is called by pressing the Task.
